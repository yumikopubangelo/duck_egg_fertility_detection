3.3   Formula Matematika yang Digunakan

1. Contrast Limited Adaptive Histogram Equalization (CLAHE)

CLAHE membagi citra menjadi blok-blok lokal (tiles), kemudian melakukan ekualisasi histogram dengan pembatasan kontras untuk menghindari amplifikasi noise berlebihan.

Fungsi transfer ekualisasi histogram: \[ s_k = T(r_k) = \sum_{j=0}^{k} \frac{n_j}{N} \]

Pembatasan kemiringan (clip limit) pada setiap nilai intensitas: \[ h_{\text{clip}}(r) = \min\!\left(h(r),\; \beta_{\text{clip}}\right), \qquad \beta_{\text{clip}} = \frac{N_{\text{pixels}}}{L} + \frac{\text{excess}}{L} \]

SimbolKeterangan
\(s_k\)Nilai intensitas output pada level \(k\)
\(r_k\)Nilai intensitas input pada level \(k\)
\(n_j\)Jumlah piksel pada level intensitas \(j\)
\(N\)Total jumlah piksel dalam tile
\(\beta_{\text{clip}}\)Batas maksimum kemiringan histogram
\(L\)Jumlah level intensitas (misal 256)

2. Homomorphic Filtering

Citra \(f(x,y)\) dimodelkan sebagai perkalian komponen iluminasi \(i(x,y)\) dan reflektansi \(r(x,y)\):

\[ f(x,y) = i(x,y) \cdot r(x,y) \]

Dengan transformasi logaritmik dan filter frekuensi tinggi \(H(u,v)\): \[ \hat{z}(x,y) = \mathcal{F}^{-1}\!\left\{H(u,v) \cdot \mathcal{F}\!\left\{\ln f(x,y)\right\}\right\} \]

Filter Butterworth orde-\(n\) untuk homomorphic: \[ H(u,v) = \left(\gamma_H - \gamma_L\right)\left[1 - \frac{1}{1 + \left(\frac{D(u,v)}{D_0}\right)^{2n}}\right] + \gamma_L \]

SimbolKeterangan
\(\gamma_H, \gamma_L\)Gain tinggi dan rendah (pengontrol kontras)
\(D(u,v)\)Jarak titik \((u,v)\) dari pusat spektrum
\(D_0\)Frekuensi cutoff
\(n\)Orde filter

3. Fungsi Loss U-Net (Binary Cross-Entropy + Dice Loss)

Binary Cross-Entropy Loss: \[ \mathcal{L}_{\text{BCE}} = -\frac{1}{N}\sum_{i=1}^{N}\left[y_i \log \hat{y}_i + (1-y_i)\log(1-\hat{y}_i)\right] \]

Dice Loss (berdasarkan koefisien Sørensen–Dice): \[ \mathcal{L}_{\text{Dice}} = 1 - \frac{2\,\sum_{i} y_i \hat{y}_i + \epsilon}{\sum_{i} y_i + \sum_{i} \hat{y}_i + \epsilon} \]

Total loss (kombinasi): \[ \mathcal{L}_{\text{total}} = \mathcal{L}_{\text{BCE}} + \mathcal{L}_{\text{Dice}} \]

SimbolKeterangan
\(y_i\)Label ground truth piksel ke-\(i\) (0 atau 1)
\(\hat{y}_i\)Prediksi probabilitas segmentasi piksel ke-\(i\)
\(\epsilon\)Nilai kecil untuk menghindari pembagian nol (misal \(10^{-6}\))

4. Ekstraksi Fitur Tekstur – GLCM (Gray Level Co-occurrence Matrix)

Elemen matriks GLCM pada arah \(\theta\) dan jarak \(d\): \[ P(i,j \mid d,\theta) = \frac{C(i,j \mid d,\theta)}{\displaystyle\sum_{i,j} C(i,j \mid d,\theta)} \]

Fitur-fitur yang diekstraksi dari GLCM:

Kontras: \(\displaystyle \text{Con} = \sum_{i,j}(i-j)^2 P(i,j)\)

Energi: \(\displaystyle \text{Ene} = \sum_{i,j} P(i,j)^2\)

Homogenitas: \(\displaystyle \text{Hom} = \sum_{i,j}\frac{P(i,j)}{1+|i-j|}\)

Korelasi: \(\displaystyle \text{Cor} = \sum_{i,j}\frac{(i-\mu_i)(j-\mu_j)P(i,j)}{\sigma_i \sigma_j}\)

SimbolKeterangan
\(C(i,j)\)Frekuensi kemunculan pasangan intensitas \((i,j)\)
\(\mu_i, \mu_j\)Rata-rata intensitas baris dan kolom
\(\sigma_i, \sigma_j\)Standar deviasi intensitas

5. Local Binary Pattern (LBP)

Untuk piksel pusat \((x_c,y_c)\) dengan \(P\) tetangga pada radius \(R\): \[ \text{LBP}_{P,R}(x_c,y_c) = \sum_{p=0}^{P-1} s\!\left(g_p - g_c\right)\, 2^p \] di mana fungsi tanda: \[ s(u) = \begin{cases} 1, & u \geq 0 \\ 0, & u < 0 \end{cases} \]

SimbolKeterangan
\(g_c\)Nilai intensitas piksel pusat
\(g_p\)Nilai intensitas piksel tetangga ke-\(p\)
\(P\)Jumlah titik sampling (tetangga)
\(R\)Radius lingkungan

6. Adaptive Weight Clustering (AWC)

(a) Jarak Euclidean antar sampel

\[ d(x_i, x_j) = \|x_i - x_j\|_2 = \sqrt{\sum_{k=1}^{d}(x_{ik} - x_{jk})^2} \]

(b) Massa Empiris Overlap (irisan dua bola)

\[ m^{\text{ov}}_{ij} = \#\!\left\{x_l \in \mathcal{X} \;\Big|\; x_l \in B(x_i,r_t) \cap B(x_j,r_t),\; l \neq i,j\right\} \]

(c) Massa Empiris Komplemen

\[ m^{\text{co}}_{ij} = \#\!\left\{x_l \in B(x_i,r_t) \cup B(x_j,r_t) \;\Big|\; x_l \notin B(x_i,r_t) \cap B(x_j,r_t)\right\} \]

(d) Massa Union (gabungan)

\[ m^{\text{un}}_{ij} = m^{\text{ov}}_{ij} + m^{\text{co}}_{ij} \]

(e) Rasio Kepadatan Overlap

\[ \hat{\rho}_{ij} = \frac{m^{\text{ov}}_{ij}}{m^{\text{un}}_{ij}} \] Dibandingkan dengan rasio volume teoretis: \[ \rho_0 = \frac{\text{Vol}\!\left(B(x_i,r_t) \cap B(x_j,r_t)\right)}{\text{Vol}\!\left(B(x_i,r_t) \cup B(x_j,r_t)\right)} \]

(f) Statistik Uji Likelihood Ratio (Hipotesis No-Gap)

\[ T_{ij} = 2\left[\,m^{\text{ov}}_{ij}\ln\!\frac{\hat{\rho}_{ij}}{\rho_0} + m^{\text{co}}_{ij}\ln\!\frac{1-\hat{\rho}_{ij}}{1-\rho_0}\right] \]

(g) Pembaruan Bobot

\[ w_{ij}^{(t+1)} = \begin{cases} 1, & \text{jika } T_{ij} \leq \lambda \text{ (klaster digabung)} \\ 0, & \text{jika } T_{ij} > \lambda \text{ (klaster dipisah)} \end{cases} \] di mana \(\lambda\) adalah hyperparameter sensitivitas (\(\lambda\) besar → cenderung menggabungkan klaster).

(h) Softmax Normalization Bobot Fitur

\[ w_k = \frac{e^{\alpha_k}}{\displaystyle\sum_{j=1}^{K} e^{\alpha_j}}, \qquad \sum_{k=1}^K w_k = 1 \]

SimbolKeterangan
\(r_t\)Radius neighborhood pada iterasi \(t\)
\(\lambda\)Hyperparameter sensitivitas AWC
\(w_k\)Bobot adaptif fitur ke-\(k\)
\(\alpha_k\)Skor relevansi fitur ke-\(k\)
\(K\)Total jumlah fitur

7. Metrik Evaluasi Model

Akurasi: \[ \text{Accuracy} = \frac{TP + TN}{TP + TN + FP + FN} \]

Presisi: \[ \text{Precision} = \frac{TP}{TP + FP} \]

Recall (Sensitivitas): \[ \text{Recall} = \frac{TP}{TP + FN} \]

Spesifisitas: \[ \text{Specificity} = \frac{TN}{TN + FP} \]

F1-Score: \[ F_1 = 2 \cdot \frac{\text{Precision} \times \text{Recall}}{\text{Precision} + \text{Recall}} \]

Dice Coefficient (IoU Segmentasi): \[ \text{DC} = \frac{2|A \cap B|}{|A| + |B|}, \qquad \text{IoU} = \frac{|A \cap B|}{|A \cup B|} \]

Brier Score: \[ \text{BS} = \frac{1}{N}\sum_{i=1}^{N}(\hat{p}_i - y_i)^2 \]

SimbolKeterangan
TPTrue Positive – telur fertil terdeteksi benar
TNTrue Negative – telur infertil terdeteksi benar
FPFalse Positive – telur infertil salah diklasifikasi fertil
FNFalse Negative – telur fertil salah diklasifikasi infertil
\(\hat{p}_i\)Probabilitas prediksi model untuk sampel ke-\(i\)
\(A, B\)Himpunan piksel prediksi dan ground truth (segmentasi)
