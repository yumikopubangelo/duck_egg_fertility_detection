## Migration Guide

### From 0.9.0 to 1.0.0

1. Update dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Migrate database:
   ```bash
   alembic upgrade head
   ```

3. Update configuration files

### From 0.1.0 to 0.9.0

1. Install new dependencies
2. Update data preprocessing scripts
3. Retrain models with new data
